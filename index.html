Políticas de Privacidad

En WRAPY, la privacidad de nuestros usuarios es nuestra máxima prioridad. Esta aplicación ha sido diseñada bajo el principio de "Privacidad por Defecto", asegurando que tus datos nunca salgan de tu control.

Resumen Ejecutivo: WRAPY es una aplicación 100% local. No utilizamos servidores en la nube para almacenar tus mensajes, no requerimos registros de cuenta y no tenemos acceso a tus conversaciones privadas.

1. Recopilación y Uso de Datos

WRAPY funciona interceptando las notificaciones de sistema de las aplicaciones que el usuario selecciona (como WhatsApp, Telegram, etc.). Este proceso se realiza únicamente para detectar mensajes y archivos multimedia que han sido eliminados por el remitente y guardar una copia de respaldo para el usuario.

2. Almacenamiento 100% Local

Toda la información recuperada, incluyendo textos, imágenes, videos y notas de voz, se almacena exclusivamente en la memoria interna de tu dispositivo móvil. WRAPY no cuenta con bases de datos remotas ni servicios de sincronización en la nube (Cloud).

3. Registro y Cuentas

Nuestra aplicación no requiere la creación de una cuenta de usuario, dirección de correo electrónico o número de teléfono para funcionar. No recopilamos identificadores personales que puedan vincular los datos almacenados con una identidad real.

4. Permisos Requeridos

Para su correcto funcionamiento, la app solicita los siguientes permisos:

  • Acceso a Notificaciones: Necesario para leer y respaldar los mensajes entrantes en tiempo real.
  • Acceso al Almacenamiento: Necesario para guardar las imágenes y videos recuperados en una carpeta local de tu teléfono.

5. Publicidad y Servicios de Terceros

WRAPY utiliza Google AdMob para mostrar anuncios y mantener la aplicación gratuita. Estos servicios pueden recopilar identificadores de publicidad anónimos proporcionados por Android para mostrar anuncios relevantes. Sin embargo, el contenido de tus mensajes nunca es compartido con estos proveedores ni con ninguna otra entidad.

6. Compras Integradas (Versión PRO)

Para los usuarios que adquieren la versión PRO, utilizamos el sistema de facturación oficial de Google Play Store. No almacenamos datos de tus tarjetas de crédito o métodos de pago, ya que estos son gestionados de forma segura por Google.

7. Cambios en esta Política

Podemos actualizar nuestras Políticas de Privacidad periódicamente. Te recomendamos revisar esta página para estar al tanto de cualquier cambio. El uso continuado de la aplicación implica la aceptación de estos términos.
